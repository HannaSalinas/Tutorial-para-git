Tutorial de Git

Paso 1: Inicialización del Repositorio

Para comenzar a usar Git, primero debemos inicializar un repositorio local en nuestro proyecto. Esto se hace con el siguiente comando:

git init

Este comando crea un repositorio Git vacío en tu carpeta de proyecto, lo que te permite empezar a rastrear cambios.

Paso 2: Creación de Ramas (Branch)

Las ramas permiten desarrollar nuevas funcionalidades sin afectar el código principal. Para crear una nueva rama usamos:

git checkout -b nombre-rama

Esto creará una nueva rama y te moverá a ella. Aquí un ejemplo:

git checkout -b feature/master

Paso 3: Hacer Commits

Un commit guarda los cambios realizados en tu proyecto. Para hacer un commit, primero añade los archivos al área de preparación:

git add .

Luego, confirma los cambios con:

git commit -m "Mensaje descriptivo"

Paso 4: Conectar a un Repositorio Remoto y Subir a GitHub

Para conectar tu repositorio local con GitHub, debes agregar un remoto. Usa este comando, reemplazando el enlace por el de tu repositorio:

git remote add origin https://github.com/tu-usuario/tu-repo.git

Luego, sube los cambios a GitHub con:

git push -u origin nombre-rama
